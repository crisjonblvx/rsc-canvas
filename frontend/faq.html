Canvas Setup

A Canvas API token is like a password that lets ReadySetClass communicate with Canvas on your behalf. Here's how to get one:

  1. Log into Canvas at your school's URL
  2. Click your profile picture (top right corner) → Settings
  3. Scroll down to Approved Integrations
  4. Click + New Access Token
  5. Name it "ReadySetClass" and click Generate Token
  6. Copy it immediately — Canvas only shows it once
If you lose the token, just delete it in Canvas Settings and generate a new one. It only takes 30 seconds.

It's the web address you use to log into Canvas at your school. It looks like:

  • https://yourschool.instructure.com
  • https://canvas.yourschool.edu
  • https://yourschool.canvas.com

Just look at your browser's address bar while logged into Canvas — copy everything up to (and including) the domain, with no trailing slash.

The most common causes of a failed connection:

  • Token copied incorrectly — make sure there are no extra spaces or characters. Try copying it again.
  • Token expired or revoked — go to Canvas Settings and generate a new token.
  • Wrong Canvas URL — double-check the URL by logging into Canvas and copying the address bar.
  • Token lacks permissions — when creating the token in Canvas, leave the "Expires" field blank for full access.

AI Grading

When students submit assignments to Canvas, ReadySetClass downloads each submission, reads the content, and grades it against your rubric using AI. Here's the full flow:

  1. Go to ✨ AI Grader in the sidebar
  2. Select the assignment with ungraded submissions
  3. Confirm your rubric (imported from Canvas automatically, or type one in)
  4. Click Start AI Grading — AI grades all submissions in 3–5 minutes
  5. Review each grade, edit scores and feedback as needed
  6. Click Post to Canvas — grades go live in your Canvas gradebook
AI is your assistant, not your replacement. Every grade is reviewed and approved by you before students see it.

ReadySetClass can read and grade submissions in these formats:

  • PDF (.pdf)
  • Word documents (.docx)
  • Excel spreadsheets (.xlsx)
  • Plain text (.txt)
  • CSV files (.csv)
  • Canvas online text submissions (typed directly in Canvas)
Submissions that are only a URL (website link) will be flagged for manual review since AI can't browse the web.

Accuracy depends on the clarity of your rubric. With a detailed rubric, AI grading is highly consistent — often more consistent than human grading across 20+ submissions where fatigue becomes a factor. That said:

  • AI grades may need adjustment for nuanced, subjective criteria
  • Low-confidence grades are flagged so you know which ones to look at more carefully
  • You review and edit every grade before posting

Think of it like a very fast first pass — you do the final call.

There is an optional AI content check flag in the grading setup. When enabled, the AI will note if a submission has patterns commonly associated with AI-generated text and flag it for your review. This is not a definitive detection tool — treat it as an indicator that warrants a closer look, not a verdict.

Privacy & Security

ReadySetClass is designed with FERPA considerations in mind. Student submission content is processed only for the purpose of generating grades — it is not stored permanently, sold, or used for any purpose other than providing the grading result to you. We recommend reviewing our privacy policy and, if required by your institution, completing a data processing agreement before use in credit-bearing courses.

Your Canvas API token is encrypted using AES-256 symmetric encryption (Fernet) before being stored in our database. It is never logged, displayed in plaintext after submission, or shared with third parties. Only ReadySetClass servers use it to make requests to Canvas on your behalf.

ReadySetClass uses your API token to access the courses and assignments you interact with in the app. It can technically read any course you have access to in Canvas (since the token has your full permissions), but it only requests data for courses you actively select. You can revoke the token at any time from your Canvas Settings to immediately cut off access.
